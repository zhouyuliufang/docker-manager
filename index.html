  • 🐋 容器管理：启停 / 重启 / 删除 / 实时日志
  • 🏪 应用商店：60+ 精选应用，分类展示，一键部署
  • 🚀 Compose 部署：在线编辑、格式验证、一键启动
  • 🔐 密码认证：首次登录设置密码，JWT Token 保护

📖 使用手册

🚀 快速开始

  • 首次访问时会要求设置访问密码（至少 6 位），设置后永久生效
  • 登录后点击顶栏 连接 按钮加载容器列表
  • API 地址留空时使用内置代理（推荐，无需额外配置）

🐋 容器管理

  • ▶ 启动：启动已停止的容器
  • ⏹ 停止：优雅停止运行中的容器
  • 🔄 重启：重启容器（停止后自动启动）
  • 📋 日志：实时查看容器输出，支持自动滚动
  • ⬆ 更新：拉取最新镜像后重新创建容器
  • 🗑 删除：删除容器（数据卷默认保留）

🏪 应用商店

  • 按分类浏览 60+ 精选 Docker 应用
  • 支持关键词搜索和分类筛选
  • 点击 一键部署 自动填入 Compose 配置并跳转到部署页面
  • 部署前可修改端口、数据卷路径等配置

🚀 Compose 部署

  • 选择快速模板或在编辑器中粘贴 docker-compose.yml
  • 点击 验证 检查 YAML 格式是否正确
  • 填写项目名（可选），点击 一键部署
  • 部署日志实时滚动显示在编辑器下方
  • 通过 停止并移除项目 停止已部署的项目

🔑 安全设置

  • 可在右上角用户菜单中 修改密码
  • Token 有效期 7 天，过期后自动跳回登录页
  • 建议密码 8 位以上，包含字母+数字

⚠️ 注意事项

  • 删除容器操作不可恢复，请谨慎操作
  • 数据卷路径建议使用绝对路径，如 /data/Docker/应用名
  • 同时删除数据卷后数据将永久丢失，请提前备份
